There are several different quantities that need to be defined. These include:
\begin{itemize}

	\setlength{\parskip}{-6pt}%
	\item Thrust to weight ratio
	\item Mass
	\item Dimensions
	\item Power requirements
	\item Flight duration
	\item Noise levels
	\item Water proofing 
\end{itemize}
